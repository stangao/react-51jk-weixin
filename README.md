# react-cnode
基于webpack + react + react-router + redux + less + flex.css + ES6 的React版微信



### 下载
```
  git clone git@git.local.ybzf.com:frontend/weixin.git
  cd weixin
  npm install (安装依赖模块)
  npm install webpack -g (没有安装webpack的需要安装)
```

### 运行（nodejs 6.0+）
```
  npm run dev (开发版本访问：http://localhost:3000/)
  npm run dist （发布生产版本）

```
### 功能
```
  1.登录退出
  2.首页
  3.分类
  4.购物车
  5.个人中心
  6.订单
```
